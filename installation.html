Installation¶

Stable release¶

To install suxarray, run this command in your terminal:

$ pip install suxarray

This is the preferred method to install suxarray, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for suxarray can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/cadwrdeltamodeling/suxarray

Or download the tarball:

$ curl -OJL https://github.com/cadwrdeltamodeling/suxarray/tarball/main

Once you have a copy of the source, you can install it with:

$ python setup.py install
